Install pytorch
Install pytorch_lightning
Install pytorch-lightning-bolts
Install lightning-flash
Install https://github.com/CeadeS/ImageZipDataset

To run the experiment:

NUM_NODES is the number of Machines
SRV_HOSTNAME is the name of the main server
If you have not 8 GPU per Machine, change --nproc_per_node in the*.sh files to the number of GPU you have

You can load the created model to extract features.
